小菜鸟用github搭建博客的总结

MikeKing-Max

2019-07-15

#首先安装node.js

#然后注册github账号（支持网易、QQ邮箱），

#然后安装git，服务器搭建网站不用安装git这一步（因为我们的vps主机在github，然后我们的本地的网站要和github上我们上传的网站建立联系要用到git，而如果我们是自己的服务器的话就不需要git了）

#再就是github的ssh钥匙配置，这个ssh密钥就类似于服务器的密码一样，这是管理员访问网站的密码，是管理员私有的要是凭证

#然后建立一个文件夹（默认名称可以设置为blog，默认安装路径在C盘，方便查找）然后打开blog，打开cmd或者git（bash）（还是不明白为什么安装了git之后那些不同的exe应用程序的单独的具体作用），键入新建的文件夹，安装hexo（hexo是网站的框架，一串串的代码构成了美丽的网站），

#安装sublime，方便打开内部的代码，类似于安装python可以读取python语言，安装sublime可以打开hexo网站框架的代码并且进行修改，

#再就是利用下载的sublime对原始的hexo网站框架代码进行修改（添加github给我们的默认的域名，修改网站样式主题为next等等），可以修改为将本地的网站上传到github，然后就可以通过浏览器进行访问了

#其实在代码的世界里，包含的是一串串字符，在后台的代码并不能像前台加了壳的应用一样随意使用，写博客不能像是在office word软件中那样自由，要按照规则比如说简明扼要又指明性强的markdown语言是挺麻烦的，这时候我们写博客并上传可以借用插件Hexo Admin，安装代码npm install –save hexo-admin，很方便修改本地的网站内容使之更完美后，敲入hexo d代码，将修改过的网站内容上传到github上，这样通过网络浏览器访问的在线网站也就修改过了更完美了

#我是个小菜鸡，现在简单的会使用的代码是hexo d、hexo clean（听说哟应用不过好像挺鸡肋的）、hexo s -p 5555、hexo s -d（好像是前两者的结合版）

#就这些了
